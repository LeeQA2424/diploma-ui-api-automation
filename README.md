#diploma-ui-api-automation
#UI + API Автотесты для сайта Читай-город

#Описание проекта

Данный проект представляет собой автоматизацию UI- и API-тестирования интернет-магазина книг **Читай-город**.

Проект реализован в рамках дипломной работы по курсу "Автоматизация тестирования".

В рамках работы были:
-разработаны UI-тесты на основе чек-листа ручного тестирования
-реализованы API-тесты на основе запросов и сценариев
-настроена структура проекта с использованием Page Object Model
-добавлена интеграция с Allure для генерации отчетов


#Цель тестирования

Основная цель — проверить корректность работы ключевого функционала сайта:

-поиск товаров
-отображение результатов поиска
-открытие карточки товара
-обработка некорректных запросов
-корректность API-ответов


#Область тестирования

#UI тесты:
-проверка наличия поля поиска
-поиск книги
-поиск с пустым запросом
-поиск с некорректным запросом
-открытие карточки товара

#API тесты:
-поиск книги через API
-пустой запрос
-некорректный запрос
-проверка структуры ответа
-проверка невалидного endpoint
-проверка авторизации


#Используемые технологии

-Python 3.14
-pytest
-selenium
-requests
-allure-pytest 
#Установка заисимоситей

pip install -r requirements.txt

#Запуск тестов

Все тесты
pytest

 Только UI тесты
pytest -m ui

Только API тесты
pytest -m api

#Генерация отчета
pytest --alluredir=allure-results
allure serve allure-results

#Проверка на ошибки
flake8
